# ssm_edu_system
SSM架构演示-教育后台管理系统

## 后端技术
- Web层：SpringMVC接收请求，进行视图跳转
- Service层：Spring进行IOC、AOP、事务管理
- dao层：Mybatis进行数据库交互

## 项目开发环境
- 开发工具：后端->IDEA、数据库->Navicat Premium
- 开发环境：JDK11、Maven 3.6.3、MySQL 5.7
